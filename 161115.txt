Користени команди:
-Откако го креирав репозиториумот и додадов README file, го клонирав користејќи ја командата git clone..
-За додавање на датотеката со 5 производи:
notepad shopping_list.txt
git add . ( најпрво ги и додаваме промените)
git commit -m "Add 5 items to shopping list" (потоа правиме commit)
-На ист начин како и претходно ги направив промените во датотеката и ги додадов количините на производите.
Потоа, користејќи git push сите commits кои ги направив локално се пренесуваат на remote repository.
-Креирање нови гранки:
git branch branch1; git branch branch2 -со оваа команда само ги креирав новите гранки,а за подоцна да се 
префрлам на нив ја користев командата git checkout branch1/branch2.
-На ист начин како и претходно ги направив промените на branch1. Тука може да се забележи дека промената на
податоците во таа гранка нема никакво влијание врз податоците во master гранката, односно тие 
остануваат непроменети. Потоа се префрлив на master гранката каде што направив промена, па потоа на 
branch2 уште една промена.
- Користејќи ја наредбата git merge ги поврзав branch1 и master, при што се промена податокот во master на
вредноста која беше во branch1, притоа не се јави никаков конфликт при извршување на наредбата.
